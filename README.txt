NOTE MAP GUITAR - VERSION 3.2

Single-file release: upload index.html to GitHub Pages.

3.2 changes:
- Added Intermediate skill level.
- Added Harmony / Tonnetz view for intermediate users.
- Tonnetz view renders a phone-readable harmony diagram:
  - yellow = selected key root
  - blue = tonic chord tones
  - green = other notes in the major key
  - horizontal lines show fifth relationships
  - diagonal lines show third relationships
- Added plain-language Tonnetz explanations underneath the diagram.

Included from 3.1:
- Early Learner mode with Chord shapes, Neck notes, and Pentatonic shapes.
- Manual mode with Scale map, Chord fingering, Progression, and Solo Blueprint.
- Context-specific guidance underneath the fretboard.
- Browser-native Web Audio playback for chords, notes, scale tones, and solo phrases.
- Validated movable chord fingerings for E/A/D-family shapes.
