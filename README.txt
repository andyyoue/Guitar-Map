NOTE MAP GUITAR - VERSION 3.6

Single-file release: upload index.html to GitHub Pages.

3.6 fixes:
- Practice Scale playback is now constrained to the selected scale pattern.
- Hear the Key remains the broader ear-training mode that highlights matching note names across the fretboard.

Included from 3.5:
- Manual > Scale Map now refreshes immediately when switching Scale action:
  - Hear the Key
  - Practice Scale
- Practice Scale playback now animates the selected fret path.
- Practice Scale uses exact string/fret cell targets, avoiding ambiguous duplicate E-string matches.
- Renamed Chord overlay to Optional chord tones.
- Optional chord tones now defaults to None and is explained as an overlay, not a scale change.

Included from 3.4:
- Hear the Key for note-location and ear training.
- Practice Scale for a single playable fingering path.
- Solo Trainer for short phrase imitation and timing.
- Corrected Tonnetz topology from 3.3.
