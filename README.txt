NOTE MAP GUITAR - VERSION 4.9

Built from 4.8.

Changes:
- Tuner microphone pipeline widened: high-pass/low-pass instead of narrow band-pass, explicit AudioContext resume, lower first-pass thresholds.
- Early Learner Chord Shapes now shows the neck view and classic chord diagram together; no diagram toggle required.
- Early Learner chord fretboard is constrained to the first five frets.
- Action buttons stay on one row above the fretboard on phone via horizontal scroll.
- Reduced repeated chord-name text in Early Learner and Progression action metadata.
- Added a Progression loop button that advances chords every 2 seconds.
