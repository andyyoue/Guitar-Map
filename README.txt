NOTE MAP GUITAR - VERSION 3.4

Single-file release: upload index.html to GitHub Pages.

3.4 changes:
- Separated "learning the notes in a key" from "practicing a scale."
- Manual > Scale Map now has a Scale action selector:
  - Hear the Key
  - Practice Scale
- Hear the Key:
  - highlights every occurrence of each scale tone across the fretboard
  - plays root-to-root
  - frames the feature as ear training plus note-location learning
- Practice Scale:
  - chooses one playable fingering path in the visible map/window
  - animates one exact fret at a time
  - frames the feature as a guitarist scale-practice drill
- Renamed Manual > Solo Blueprint to Solo Trainer.
- Solo Trainer keeps the existing short phrase playback/animation, now positioned as imitation and timing practice.

Included from 3.3:
- Corrected Tonnetz topology for major/minor triad triangles.
- Intermediate Harmony / Tonnetz view.
- Early Learner mode with Chord shapes, Neck notes, and Pentatonic shapes.
- Manual Chord fingering and Progression views.
