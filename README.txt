NOTE MAP GUITAR - VERSION 3.3

Single-file release: upload index.html to GitHub Pages.

3.3 changes:
- Fixed Harmony / Tonnetz topology so major and minor triads appear as complete connected triangles.
- Preserved the flattened left/right fifths layout and overall visual style.
- Re-centered the Tonnetz slice around the selected key so the most useful harmony relationships stay visible.
- Added missing minor-third connections.
- Clarified the explanation:
  - horizontal = fifths
  - vertical = major thirds
  - diagonal = minor thirds
  - complete triangles = major/minor triads
- Clarified that this is intended as a mathematically correct Tonnetz slice for visible major/minor triads, while still simplified for enharmonic spellings and advanced chord types.

Verified connected triangles:
- E-G#-B
- A-C#-E
- B-D#-F#
- C#-E-G#
- F#-A-C#
- G#-B-D#
- D#-F#-A#

Included from 3.2:
- Intermediate skill level.
- Harmony / Tonnetz view.
- Early Learner mode with Chord shapes, Neck notes, and Pentatonic shapes.
- Manual mode with Scale map, Chord fingering, Progression, and Solo Blueprint.
